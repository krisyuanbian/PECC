---

## Overview

This repository provides the code and data necessary to reproduce all numerical results presented in the paper. The implementation includes both parametric and Super Learner approaches for propensity score weighting in causal inference.

---

## Repository Structure

### Code

- **`code.R`**  
  Contains all scripts required to generate the numerical results and analyses reported in the paper.

### Data

- **`Ptdata.csv`**  
  A cleaned subset of the data obtained from the following case study:  
  https://ssc.ca/en/case-study/developing-a-physician-performance-model-critical-care-assessing-quality-and-value

---

## Reproducibility

Running `code.R` with the provided dataset will reproduce the results reported in the manuscript. Please ensure that all required R packages are installed prior to execution.
